### Rendering examples

Render using class

```html
<div class="eosc-common-totop-wrapper"></div>
```

Render using id

**IMPORTANT!!! Only first element with the id will be rendered!**

```html
<div id="eosc-common-totop-wrapper"></div>
```

Render using camel case

```html
<EoscCommonTotopWrapper></EoscCommonTotopWrapper>
```

Render using snake case

```html
<eosc-common-totop-wrapper></eosc-common-totop-wrapper>
```

Render with custom text label

```html
<EoscCommonTotopWrapper label="some text"></EoscCommonTotopWrapper>
```
